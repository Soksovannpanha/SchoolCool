<p dir="auto">The School Management System is a web-based application built using the MERN (MongoDB, Express.js, React.js, Node.js) stack. It aims to streamline school management, class organization, and facilitate communication between students, teachers, and administrators.</p>

<ul dir="auto">
<li>
<p dir="auto"><strong>User Roles:</strong> The system supports three user roles: Admin, Teacher, and Student. Each role has specific functionalities and access levels.</p>
</li>
<li>
<p dir="auto"><strong>Admin Dashboard:</strong> Administrators can add new students and teachers, create classes and subjects, manage user accounts, and oversee system settings.</p>
</li>
<li>
<p dir="auto"><strong>Attendance Tracking:</strong> Teachers can easily take attendance for their classes, mark students as present or absent, and generate attendance reports.</p>
</li>
<li>
<p dir="auto"><strong>Performance Assessment:</strong> Teachers can assess students' performance by providing marks and feedback. Students can view their marks and track their progress over time.</p>
</li>
<li>
<p dir="auto"><strong>Data Visualization:</strong> Students can visualize their performance data through interactive charts and tables, helping them understand their academic performance at a glance.</p>
</li>
<li>
<p dir="auto"><strong>Communication:</strong> Users can communicate effortlessly through the system. Teachers can send messages to students and vice versa, promoting effective communication and collaboration.</p>
</li>
</ul>
